## القيود المتبقية

### بسبب GitHub Pages
1. **لا يمكن إضافة رؤوس HTTP مخصصة** — يتم استخدام meta tags فقط
2. **لا يمكن تشغيل Backend** — Rate Limiting يتطلب Cloudflare Worker
3. **لا يمكن تشفير الملفات الثابتة** — HTTPS فقط عبر GitHub

### بسبب عدم وجود Auth
1. **حذف الحساب** — يتطلب Supabase Auth للتحقق من الهوية
2. **عرض "بياناتي"** — لا يمكن التحقق من صاحب البيانات بدون Auth
3. **تصحيح البيانات** — لا يمكن ضمان أن الشخص.correct هو صاحب البيانات

### بسبب عدم وجود Edge Functions
1. **التحقق من العمر** — لا يمكن التحقق من أن المستخدم فوق 16 عامًا
2. **Rate Limiting خادم** — يتطلب Worker أو Edge Function
3. **الإبلاغ إلى CNPDP** — يجب يدويًا حاليًا
